## 模块职责详解

### 1. CLI 入口层 (cli.py)

```
┌─────────────────────────────────────────────────────────────────┐
│                         cli.py                                  │
│                       命令行入口                                │
├─────────────────────────────────────────────────────────────────┤
│                                                                 │
│  命令列表:                                                      │
│  ┌─────────────┐  ┌─────────────┐  ┌─────────────┐            │
│  │    eval     │  │    list     │  │    info     │            │
│  │  执行评测   │  │ 列出算子/用例│  │ 显示算子信息│            │
│  └─────────────┘  └─────────────┘  └─────────────┘            │
│  ┌─────────────┐  ┌─────────────────────────────────────────┐ │
│  │   config    │  │            eval-process                 │ │
│  │  配置管理   │  │        进程池子进程执行                  │ │
│  └─────────────┘  └─────────────────────────────────────────┘ │
│                                                                 │
│  调用链:                                                        │
│  eval → Evaluator → process_pool/subprocess_runner             │
│  list → CaseLoader/OperatorLoader                              │
│  info → OperatorLoader                                         │
│                                                                 │
└─────────────────────────────────────────────────────────────────┘
```

### 2. 评测核心层 (eval/)

```
┌─────────────────────────────────────────────────────────────────────────────┐
│                           eval/ 模块架构                                    │
├─────────────────────────────────────────────────────────────────────────────┤
│                                                                             │
│   ┌─────────────────────────────────────────────────────────────────────┐  │
│   │                       Evaluator (evaluator.py)                      │  │
│   │                    综合评测调度器                                    │  │
│   │                                                                     │  │
│   │   evaluate_case(case, golden_func)                                 │  │
│   │       ┌─────────────────────────────────────────────────┐          │  │
│   │       │ 1. 生成输入张量 (DataGenerator)                   │          │  │
│   │       │ 2. 构建调用参数 (ParamBuilder)                   │          │  │
│   │       │ 3. 执行 Golden (OpRunner.run_golden)            │          │  │
│   │       │ 4. 执行 AI 算子 (OpRunner.run_operator)         │          │  │
│   │       │ 5. 精度对比 (AccuracyEvaluator)                 │          │  │
│   │       │ 6. 性能采集 (PerfEvaluator)                     │          │  │
│   │       │ 7. 生成结果 (EvalCaseResult)                    │          │  │
│   │       └─────────────────────────────────────────────────┘          │  │
│   └─────────────────────────────────────────────────────────────────────┘  │
│                                                                             │
│          ┌─────────────────────────────────────────────────────────────┐   │
│          │                    OpRunner (op_runner.py)                  │   │
│          │                      算子执行器                              │   │
│          │                                                             │   │
│          │  run_golden(func, params, ...) → CPU fp64 参考             │   │
│          │  run_operator(op_func, params, ...) → NPU 执行             │   │
│          │                                                             │   │
│          └─────────────────────────────────────────────────────────────┘   │
│                                                                             │
│   ┌──────────────────┐        ┌──────────────────┐                       │
│   │ AccuracyEvaluator │        │  PerfEvaluator   │                       │
│   │   (accuracy_eval) │        │   (perf_eval)    │                       │
│   │                  │        │                  │                       │
│   │  精度对比:       │        │  性能采集:       │                       │
│   │  - MERE/MARE    │        │  - Profiler      │                       │
│   │  - CPU fp64     │        │  - 升频清cache   │                       │
│   │  - 二次验证     │        │  - kernel_only   │                       │
│   └──────────────────┘        └──────────────────┘                       │
│                                                                             │
│   ┌──────────────────────────────────────────────────────────────────┐    │
│   │              ProcessPoolCoordinator (process_pool.py)             │    │
│   │                       进程池协调器                                │    │
│   │                                                                   │    │
│   │  多卡 × 多进程并行架构:                                          │    │
│   │  ┌────────────────────────────────────────────────────────────┐  │    │
│   │  │  Card 0 ─┬─ Process 0 ─→ 算子 A (case 1-20)               │  │    │
│   │  │          └─ Process 1 ─→ 算子 B (case 1-20)               │  │    │
│   │  │  Card 1 ─┬─ Process 2 ─→ 算子 C (case 1-20)               │  │    │
│   │  │          └─ Process 3 ─→ 算子 D (case 1-20)               │  │    │
│   │  └────────────────────────────────────────────────────────────┘  │    │
│   │                                                                   │    │
│   │  动态调度: 每算子一个进程，每卡并发上限                           │    │
│   └──────────────────────────────────────────────────────────────────┘    │
│                                                                             │
│   ┌──────────────────┐        ┌──────────────────┐                       │
│   │    InputPool     │        │     Results      │                       │
│   │   (input_pool)   │        │    (results)     │                       │
│   │                  │        │                  │                       │
│   │  防缓存攻击:     │        │  数据结构:       │                       │
│   │  - 预分配clone   │        │  - EvalCaseResult│                       │
│   │  - 轮换使用      │        │  - EvalOperator  │                       │
│   │                  │        │  - EvalSession   │                       │
│   └──────────────────┘        └──────────────────┘                       │
│                                                                             │
└─────────────────────────────────────────────────────────────────────────────┘
```
